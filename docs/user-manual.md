# 果评赛事 App 使用手册（全角色）

最后更新：2026-03-03

## 1. 这份手册适合谁

这份手册面向**所有app使用者**，帮助你按步骤完成：

- 注册和登录
- 参与赛事、提交作品、查看评分数据
- 作为评委进行打分
- 作为管理员管理用户、赛事、模型和水果数据

只需要按页面提示操作即可。

---

## 2. 角色与权限一览

系统包含 3 类主要角色：

| 角色 | 你可以做什么 |
|---|---|
| 普通用户 | 查看赛事、查看评分数据、提交作品、编辑/删除自己的作品、查询水果数据 |
| 评委用户 | 拥有普通用户全部能力 + 对分配到的作品进行评分 |
| 管理员 | 拥有全部能力 + 进入管理员面板，管理用户、赛事、模型、水果数据 |

说明：

- “评委”是管理员在某个赛事中分配的身份，不是单独注册的角色。
- “管理员”在首页会看到“管理员面板”入口。

---

## 3. 首次使用（注册与登录）

### 3.1 注册账号

1. 打开 App，进入登录页。
2. 点击“立即注册”。
3. 输入用户名和密码。
4. 点击“Register/注册”。

账号规则：

- 用户名：3-20 位，只能用字母、数字、下划线
- 密码：至少 8 位

### 3.2 登录账号

1. 返回登录页。
2. 输入用户名和密码。
3. 点击“登录”。
4. 登录成功后进入首页。

---

## 4. 首页功能总览

登录后首页主要有 3 个入口：

1. 品质评价
2. 水果营养查询
3. 管理员面板（仅管理员可见）

右上角可退出登录。

---

## 5. 普通用户使用流程

## 5.1 进入“品质评价”

1. 首页点击“品质评价”。
2. 查看赛事列表。
3. 点击某个赛事卡片展开操作。

你会看到常用操作：

- 数据：查看该赛事作品评分概览
- 我的作品：查看和管理自己的作品
- 提交：提交新作品（仅进行中的赛事显示）

## 5.2 提交参赛作品

路径：`品质评价 -> 选择赛事 -> 提交`

步骤：

1. 填写作品名称（必填）。
2. 填写作品简介（可选）。
3. 选择作品图片（可选）。
4. 点击“提交”。

提交后可在“我的作品”中查看。

## 5.3 管理“我的作品”

路径：`品质评价 -> 选择赛事 -> 我的作品`

你可以：

- 编辑作品名称/简介
- 删除自己的作品
- 查看作品审核状态（待审核/已通过/已拒绝）

## 5.4 查看赛事评分数据

路径：`品质评价 -> 选择赛事 -> 数据`

列表页会显示：

- 作品名称
- 参赛者
- 均分

点击某个作品可进入详情页，查看：

- 作品图片
- 评分雷达图
- 权重占比图
- 平均分、最高分、评分人数
- 各评分维度分数

## 5.5 查询水果数据

路径：`首页 -> 水果营养查询`

步骤：

1. 选择“数据类型”。
2. 选择“水果”。
3. 点击“查询”。
4. 在结果区查看对应数据。

---

## 6. 评委使用流程

评委在普通用户流程基础上，增加“评分”能力。

## 6.1 进入评分页面

常见路径：

- `品质评价 -> 数据 -> 作品详情 -> 开始评分`

## 6.2 评分操作

在评分页面中：

1. 每个维度通过滑块打分。
2. 可填写评语（可选）。
3. 点击“提交评分”或“更新评分”。

提交成功后会提示“评分提交成功”。

提示：若无法评分，通常是因为该赛事未将你分配为评委。

---

## 7. 管理员使用流程

路径：`首页 -> 管理员面板`

管理员面板包含：

1. 模型管理入口
2. 用户管理入口
3. 赛事管理入口
4. 水果数据管理

## 7.1 用户管理（新增能力）

路径：`管理员面板 -> 用户管理入口`

你可以：

- 查看全部用户列表
- 创建新用户
- 切换用户权限：普通用户 <-> 管理员
- 删除用户

安全限制：

- 当前登录管理员不能删除自己
- 当前登录管理员不能把自己降为普通用户
- 系统必须至少保留 1 个管理员

## 7.2 模型管理

路径：`管理员面板 -> 模型管理入口`

你可以：

- 新建评价模型
- 编辑模型（名称、描述、评分维度）
- 删除模型

注意：模型维度权重总和通常应为 100%。

## 7.3 赛事管理

路径：`管理员面板 -> 赛事管理入口`

你可以：

- 添加赛事
- 编辑赛事（名称、介绍、截止日期、评价模型）
- 管理评委（已有评委列表 + 搜索用户添加）
- 删除赛事
- 审核作品（通过/待审/拒绝）
- 编辑作品

### 7.3.1 添加赛事

步骤：

1. 填写赛事名称。
2. 选择截止日期。
3. 选择评价模型。
4. 选择评委（可选）。
5. 点击“添加赛事”。

### 7.3.2 编辑赛事中的评委管理

在“编辑赛事”页面：

- 上半区显示“已有评委”，可直接移除
- 下半区使用搜索框查找用户并“添加”为评委

## 7.4 作品审核

路径：`赛事管理 -> 某赛事 -> 审核作品`

你可以对每个作品设置状态：

- 通过
- 待审核
- 拒绝

## 7.5 水果数据管理

路径：`管理员面板 -> 水果数据管理`

分为两个标签页：

1. 字段管理
   - 添加/删除数据类型
   - 添加/删除水果种类
2. 表格管理
   - 选择数据类型 + 水果
   - 查询是否有数据表
   - 上传 CSV 创建表格
   - 删除表格

---

## 8. 推荐的完整操作顺序（管理员开新赛）

如果你是第一次开赛，建议按这个顺序：

1. 在“模型管理”先创建评价模型。
2. 在“用户管理”确认参赛用户、评委用户已创建。
3. 在“赛事管理”新增赛事并分配评委。
4. 通知用户在“品质评价”中提交作品。
5. 在“审核作品”里处理作品状态。
6. 评委进入作品详情开始评分。
7. 在“数据”页查看均分和图表统计。

---

## 9. 常见问题

## 9.1 为什么看不到“管理员面板”？

你的账号不是管理员。请联系管理员在“用户管理”中提升权限。

## 9.2 为什么看不到“提交”按钮？

该赛事可能不是“进行中”，或已过截止日期。

## 9.3 为什么无法评分？

你可能未被添加为该赛事评委。请让管理员在“编辑赛事”里添加你。

## 9.4 为什么不能删除某个用户？

可能触发系统安全限制（例如删除当前登录管理员，或会导致管理员数量为 0）。

## 9.5 提交图片失败怎么办？

建议先检查：

1. 网络是否正常
2. 图片是否可正常打开
3. 重试一次提交
